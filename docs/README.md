## 🎯 Sobre o Projeto

O site da **FHD Automação Industrial** é uma aplicação web moderna desenvolvida com:

- **Frontend**: React + Vite
- **Backend**: Supabase (PostgreSQL)
- **Deploy**: Vercel
- **UI**: Tailwind CSS + Shadcn/ui
- **Autenticação**: Supabase Auth
